📒 《 关于我用拓展运算符把项目搞崩这件事 》

注意函数调用不能接受过长的参数：

```ts
const items = [];
const newItems=new Array(1000000)

// ❎ 以下两种情况，如果 newItems 很长容易导致爆栈问题
items.push(...newItems);
items.push.apply(items, newItems);

// ✅ 合理做法
items.concat(newItems);
items = [...items, ...newItems];
```
