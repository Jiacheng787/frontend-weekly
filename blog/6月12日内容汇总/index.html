📒 NPM 发包脚本加上下面这段代码

const npmPublish = async () => {
  const scripts = [
    "npm set-script postinstall \"\"",
    "npm publish",
    "npm set-script postinstall \"patch-package\"",
  ];
  await scripts.reduce(
    (prev, script) => prev.then(() => runScript(script)),
    Promise.resolve()
  );
}

这里的 reduce() 确保了 Promise 按顺序调用，而 Promise.all() 并发执行不保证顺序
